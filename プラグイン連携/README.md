# NotebookLM 連携 — プラグイン版（NotebookLM ランチャー）

kintone の一覧画面に、**NotebookLM へ飛ぶボタン付きカード**を表示する kintone プラグインです。

- レコードや添付ファイルは **NotebookLM 側に格納されている前提** のため、
  **レコード・フィールドは一切参照しません**。
- 表示するカードの「タイトル」と「URL」は **プラグインの設定画面** で登録します。
- **既定は1カード**。設定画面の「＋カードを追加」で**複数カード**に拡張できます。
- 同じプラグインを複数アプリに適用でき、**水平展開**が可能です。

## ディレクトリ構成

```
プラグイン連携/
├── manifest.json          … プラグイン定義
├── html/
│   └── config.html        … 設定画面
├── js/
│   ├── config.js          … 設定画面ロジック（保存・行追加/削除）
│   └── notebook-cards.js  … 一覧へのカード描画（PC/モバイル共通）
├── css/
│   ├── style.css          … カード・ボタンのスタイル
│   └── config.css         … 設定画面のスタイル
├── image/
│   └── icon.png           … プラグインアイコン（48×48）
└── README.md
```

## ビルド（.zip 化）

kintone へ読み込むには `.zip` 形式にパッケージングします。Node.js 環境で
[@kintone/plugin-packer](https://github.com/kintone/js-sdk/tree/main/packages/plugin-packer) を使います。

```bash
# 親ディレクトリ（NotebookLM連携/）で実行

# 初回（秘密鍵を自動生成）
npx @kintone/plugin-packer ./プラグイン連携 --out ./プラグイン連携/plugin.zip

# 2回目以降（初回に生成された鍵を指定して更新版を作る）
npx @kintone/plugin-packer ./プラグイン連携 \
  --ppk ./プラグイン連携/aihedlomdgnddicaakbbemjcabmgnpnn.ppk \
  --out ./プラグイン連携/plugin.zip
```

- 初回実行で、プラグインID名の秘密鍵
  `aihedlomdgnddicaakbbemjcabmgnpnn.ppk` が生成されます（生成済み）。
  **この鍵は必ず保管してください**（更新時に同一プラグインとして認識させるために必要）。
- 生成された `plugin.zip` を kintone にアップロードします。
- ⚠️ `*.ppk` を Git 管理する場合は誤公開に注意してください（`.gitignore` 推奨）。
- なお `homepage_url` 未設定の WARN が出ますが、必須ではないため無視して問題ありません。

## kintone への導入

1. **システム設定** →「その他の設定」→「プラグイン」→「読み込む」で `plugin.zip` を追加
2. 対象アプリ →「アプリの設定」→「プラグイン」→「プラグインの追加」で本プラグインを追加
3. プラグイン行の **歯車（設定）** を開く

## 設定画面の使い方

1. **タイトル**（例: `2026年6月度 定例議事録`）と **NotebookLM URL**
   （`https://notebooklm.google.com/...`）を入力
2. カードを増やす場合は **「＋カードを追加」** をクリックして行を追加
3. 不要な行は各行の **「削除」** で除去
4. **「保存」** をクリック → アプリを **「アプリを更新」**

> URL は `https://` で始まる必要があります（不正な値は保存時に警告されます）。

## 仕様メモ

- 表示先は一覧の**ヘッダースペース**（`getHeaderSpaceElement`）。標準のレコード一覧はそのまま残ります。
- カードは設定配列 `cards: [{title, url}, ...]` を読み、件数分を縦並びで描画します（冪等）。
- セキュリティのため `https://` 始まり以外の URL はリンクとして無効化されます。
- アイコンはインライン SVG（sparkle）で描画し、追加画像のアップロードは不要です。
- カードが多数になった場合は、`css/style.css` の `.notebook-card-list` をグリッド表示に
  変更することで横並びにも拡張できます。
