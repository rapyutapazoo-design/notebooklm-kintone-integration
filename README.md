# NotebookLM × kintone 連携

kintone のレコード一覧から、対応する **NotebookLM** のノートをワンクリックで開くための
連携実装をまとめたリポジトリです。実装方式の違いで **2 種類** を用意しています。

| 連携 | 実装方式 | 主な用途 | フォルダ |
|---|---|---|---|
| **カスタマイズ連携** | kintone の JavaScript / CSS カスタマイズ | 単一アプリにすばやく適用 | [`カスタマイズ連携/`](カスタマイズ連携/README.md) |
| **プラグイン連携** | kintone プラグイン（設定画面付き） | 複数アプリへ水平展開 | [`プラグイン連携/`](プラグイン連携/README.md) |

## どちらを使うか

- **カスタマイズ連携**: レコードの `notebook_url`（リンク）と `title` を参照し、一覧に
  「タイトル＋ボタン」のカードを表示します。レコードごとに異なるノートへ遷移できます。
  1 つのアプリに JS / CSS をアップロードするだけで導入できます。
- **プラグイン連携**: レコードやフィールドは参照せず、**プラグイン設定画面**で登録した
  「タイトル＋URL」のカードを一覧に表示します（既定 1 枚、追加で複数枚）。プラグインとして
  パッケージングするため、複数アプリへ同じものを配布・水平展開できます。

```
NotebookLM連携/
├── カスタマイズ連携/      … JS/CSS カスタマイズ版（単一アプリ向け）
│   ├── カスタマイズ連携.js
│   ├── カスタマイズ連携.css
│   └── README.md
├── プラグイン連携/        … kintone プラグイン版（水平展開向け）
│   ├── manifest.json
│   ├── html/ js/ css/ image/
│   └── README.md
└── README.md             … 本ファイル
```

導入手順・設定方法は、各フォルダの README を参照してください。

## 共通の前提（NotebookLM 側）

- 開きたいノートは、あらかじめ NotebookLM で作成しておきます。
- ノートの URL（`https://notebooklm.google.com/...`）を、各連携の方法で登録します。
- Google Workspace アカウントのノートは、**同一ドメイン内**でのみ共有できます。
  一覧のボタンを使う利用者には、事前にノートを共有しておいてください。

## セキュリティ / 取り扱い注意

- プラグインの**秘密鍵 `*.ppk` はリポジトリにコミットしません**（`.gitignore` で除外）。
  プラグイン更新時に必要なため、ローカルで安全に保管してください。
- ビルド成果物 `plugin.zip` と OS 生成ファイル（`.DS_Store`）も `.gitignore` で除外しています。
- リンクは `https://` 始まりのみ有効化し、`javascript:` 等は無効化しています。
